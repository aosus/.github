# مجتمع أسس

## أكبر مجتمع عربي للمصادر الحرة والمفتوحة


مجتمع أسس يهدف لنشر وإثراء ثقافة المصادر الحرة والمفتوحة في العالم العربي, ولتسهيل المساهمة والمشاركة فيها
